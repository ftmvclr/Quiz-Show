Quiz Show\n
This is a customizable quiz show game.\n
All that is needed to be done is in the allStuff.txt file.\n
Place a "." before typing your question in that 1 line.\n
Type in 4 options in 4 lines.\n
Then, in the next line type the correct answer as it is in the option.\n
Without leaving empty lines in between, one could continue adding more questions.\n
